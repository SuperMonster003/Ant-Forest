### 使用说明
******
1. 下载"Ant_Forest_Project"目录  
2. 将目录中的全部文件 (或目录本身) 放置于手机存储    
3. 使用"Auto.js"运行"Ant_Forest_Launcher.js"文件  
4. 欢迎使用并反馈
> 手机存储目录推荐"Auto.js"默认可识别目录 (如 `/sdcard/Scripts/` 或 `/sdcard/脚本/` )
  
> 安卓系统最低版本: `7.0` / `Android Nougat` / `Android N`  
> Root: `可能需要Root` `(未Root设备未做运行测试)`
  
> "Auto.js"已测试版本:   
>> `4.1.0 alpha 5`  
>> `4.1.1 alpha 2`    
>> `Pro 7.0.0-3`  
>> `Pro 7.0.0-4`  
>> `Pro 7.0.0-5`  
>> `Pro 7.0.0-6` `(推荐使用)`

>"Auto.js"不适用版本:  
>> `Pro 7.0.0-7` `(运行后强制退出)`
